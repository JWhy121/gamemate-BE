### 프로젝트 목적
* 유저가 다양한 게임 모임을 생성/참여하여 커뮤니티를 이룸
* 사용자가 선호 카테고리, 플레이 시간대를 선택하여 그것을 바탕으로 비슷한 유저를 추천함
* 채팅창을 통한 실시간 소통과 편리한 의사소통을 가능하게 함
